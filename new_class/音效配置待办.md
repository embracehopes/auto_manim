|------|----------|------|

## 📝 使用说明

1. 运行 `sound_library.py` 自动创建目录结构
2. 按上表将源文件复制并重命名到 `assets/sounds/library/` 对应文件夹
3. 可使用音频编辑软件进行微调：pitch ±2%、gain ±2~4 dB
4. 持续类音效（Rotating/PhaseFlow）需手动降低音量
5. 在 AutoScene 中启用音效：
   ```python
   self.set_animation_sounds_enabled(True)
   ```
